### 这是什么？
一套基于uniapp的多平台小程序生成系统
我的目标是利用H5拖拽式界面分配组件池内部的组件，再通过EggJs执行Shell命令编译后，为用户提供可上线的小程序代码。

### 用本系统能为你做些什么
- H5拖拽式 对小白用户及其友好
- 多平台生成 包括 功能方面的兼容
- 极速生成小程序
- 内置初始模板
